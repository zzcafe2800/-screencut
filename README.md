# 便利くんG (Screencut)

アイテムのサイズに合わせた画像加工や、動画からのGIF変換、高度な画像編集などをブラウザ上ですべて完結できる多機能Webツールです。

🔗 **サイトURL:** [https://zzcafe2800.github.io/-screencut/](https://zzcafe2800.github.io/-screencut/)

---

## 💡 主な機能

* **動画からGIFへの高画質変換**：動画ファイルを読み込むだけで、簡単にアニメーションGIFを作成。
* **アイテムサイズに自動調節（オススメ君）**：画像の縦横比に近いアイテムを一致率（%）順に自動提案。
* **部屋に馴染む色自動調整（部屋にマッチ君）**：お部屋の画像に色合い・明度・暖かさを自動でマッチング。手動スライダー調整も可能。
* **自由な画像変形・背景拡張**：つかんで伸ばす変形や、スポイトによる背景色の自動拡張。
* **画像の複数分割機能**：1つの画像やGIFを、縦横自由なマス目に一括分割。

## 🚀 特徴

* **インストール不要**：ブラウザを開くだけで、ログインなしですぐに使えます。
* **ファイル容量の可視化**：容量に応じて色が変わるガイド機能付きで、SNS投稿の上限目安が直感的にわかります。
